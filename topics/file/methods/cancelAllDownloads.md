# AG.file.unzip

## Summary
`cancelAlLDownloads()` cancels all active downloads initiated by calling [AG.download](download.md).


## Quick example:
```javascript
AG.file.cancelAllDownloads();
```

## Syntax
```javascript
AG.file.cancelAllDownloads()
```

**Parameters**
No parameters.

## Returns
[AG.Deferred()](../../Deferred/Deferred.md)

## Supported platforms
* iOS
