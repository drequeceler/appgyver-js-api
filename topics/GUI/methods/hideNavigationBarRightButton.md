# AG.GUI.hideNavigationBarRightButton

## Summary
`hideNavigationBarRightButton()` hides the native right button from the navigation bar.

This method currently works only in Edge mode.

## Quick example:
```javascript
AG.GUI.hideNavigationBarRightButton();
```

## Syntax
```javascript
AG.GUI.hideNavigationBarRightButton()
```

**Parameters**

None.

## Returns
Nothing.

## Supported platforms
* iOS
