# AG.GUI.hideNavigationBarRightButton

## Summary
`hideRightButton()` hides the native right button from the navigation bar.

This method works only in normal mode.

## Quick example:
```javascript
AG.GUI.hideRightButton();
```

## Syntax
```javascript
AG.GUI.hideRightButton()
```

**Parameters**
 
None.

## Returns
Nothing.

## Supported platforms
* iOS
